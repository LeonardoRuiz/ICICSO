Reglas de integración

  • `Acompañantes` se mantiene como filtro visible, numerado y funcional.
  • Los valores se presentan en orden alfabético en la interfaz, aunque con identificadores estables para gobernanza.
  • Un recurso puede portar múltiples acompañantes cuando cambian conducta, protección, ejecución o seguimiento.
  • Los registros históricos sin base editorial suficiente conservan `No especificado` hasta revisión por lotes.
  • `AC-09` solo absorbe inmunoterapia o cardio-oncología cuando la señal biológica sea explícita y clínicamente operativa.
  • `AC-13` se aplica cuando la línea del recurso es oncológica o cardio-oncológica de forma explícita.
