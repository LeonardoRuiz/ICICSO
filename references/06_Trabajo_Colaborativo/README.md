# Trabajo Colaborativo

Este directorio existe para alojar material de trabajo en curso que todavia no califica como fuente canonica ni como historico. Su uso debe ser disciplinado para que no se convierta en una zona gris del repositorio.

## Proposito

Usar `06_Trabajo_Colaborativo/` para:

- borradores tecnicos;
- propuestas de estructura o contenido en revision;
- material compartido entre varias personas antes de consolidacion;
- artefactos intermedios que aun no pertenecen a otra capa.

## Regla de uso

Todo archivo aqui debe cumplir al menos una de estas condiciones:

1. esta en revision activa;
2. aun no esta listo para promocionarse a una capa canonica;
3. necesita validacion adicional antes de archivarse o consolidarse.

## No usar esta carpeta para

- outputs definitivos;
- fuente documental canonica;
- artefactos generados por herramientas;
- caches, logs o temporales;
- modulos de software activos.

## Destino esperado de los contenidos

Un contenido de `06_Trabajo_Colaborativo/` debe terminar en uno de estos lugares:

- `01_Arquitectura_Base/` a `05_Catalogos_Listados/` si se vuelve canon documental;
- `docs/` si se vuelve gobierno transversal del repo;
- `08_Plataforma_Digital/` o `software_orquestador/` si se convierte en software o documentacion local de software;
- `99_Archivo_Historico/` si deja de estar vigente pero debe conservarse.

## Regla de saneamiento

Si una carpeta o archivo lleva mucho tiempo sin actividad y no tiene destino claro, debe revisarse y:

- consolidarse;
- archivarse;
- o eliminarse si no aporta trazabilidad.
